# Az - Federation


# 👉 Overview

---

### 👀 What ?

Az - Federation is a service provided by Microsoft Azure for secure user authentication and authorization.


### 🧐 Why ?

Federation is important because it allows users to access multiple applications with a single sign-on, which simplifies user experience and increases security.


### ⛏️ How ?

Az - Federation can be implemented using Azure Active Directory and configuring federation with the desired applications.


### ⏳ When ?

Federation in Azure was first introduced in 2010 as a part of the Azure Active Directory service.
